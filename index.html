Project Allocation System

Allocate Project to Particular Students.

Language used:
Web Language – HTML, CSS, JavaScript
Client Side Script – Java Script
Server Side Script – PHP
Database: MySQL
Description: Project Allocation Systems are designed to manage and store project information that are used in web-based applications. By different groups of people such as, students, faculty or admin will be let by project applications a controlled access to information and automated distribution of information. The main objective of the student project allocation system is to make an internet based system which will provide information of each student to institute with ease. This project will provide an effective and efficient way to manage data with low cost. The system contains various options such as login/logout, data management by grouping same type of data, project allocation for semester, etc. which will give an interactive experience to users. It will be secure with a password so the data can be used by only those who have an id and password while the data will be maintained in the system for a long period of time.

  • Client: College Major Project
  • Category: Complete Project
